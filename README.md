## GitHub Pages setup (5 minutes, no command line)

1. Go to your repo on GitHub
2. Click **Settings** → **Pages**
3. Under Source select **Deploy from a branch**
4. Select **main** branch, **/ (root)** folder, click **Save**
5. Live in ~60 seconds at `https://coldborelabs.github.io/dope`

## Add to iPhone home screen (use Safari, not Edge)

1. Open `https://coldborelabs.github.io/dope` in **Safari**
2. Tap the Share button (box with arrow)
3. Tap **Add to Home Screen**
4. Tap **Add**
